# Projeto 4 — DCGAN no CelebA

Geração de rostos humanos coloridos 64×64 com Deep Convolutional GAN
(Radford et al., 2015), treinada no CelebFaces Attributes Dataset.

## Como rodar
1. Abrir o notebook no Kaggle.
2. Add Input → CelebFaces Attributes (CelebA) Dataset (jessicali9530).
3. Settings → Accelerator → GPU T4 x2.
4. Run All (~12-15 min com 5 epochs).
